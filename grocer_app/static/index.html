Welcome

Hello! This is the GrocerApp's server home page.

API

Use /items to get all store items, or try passing in arguments like /items?titleHas=beans. Post a new store item to /items to create a new item.

Use /lists to get all grocery lists, or try passing in arguments like /lists?storeId=2. Post a new grocery list to /lists to create a new list.

Startup

When the server has just started up, be sure to go to startup to let the server initiate the database.
